HTTP (Hypertext Transfer Protocol)
O que é HTTP? Início da década de 90; Necessidades: Transferência de Arquivos; Direcionamento e comunicação entre clientes e servidores; Solicitações de pesquisa; IP ou nome de domínio + porta 80.
Verbos HTTP O que são: Chamados de métodos, verbos e de requisições; Ações requisitadas;
Os mais conhecidos são:
GET;
POST;
HEAD;
GET:
Solicitação de dados;
Não deve ser usado para lidar com dados sensíveis/privados;
A solicitação é feita via URL.
POST:

GET: via URL;

A comunicação é feita de requests (solicitações) e responses (respostas);
Basicamente, os três pilares da requisição:
Request Line;
Header;
Body.
