    • A WEB

    • Já a World Wide Web, ou WEB para os mais íntimos, é apenas uma maneira de comunicação via internet.
    • É um metódo de navegação usando a interconexão entre textos (páginas) via hiperlinks (hoje em dia conhecidos apenas por links).

    Mas será que a WEB de hoje é a mesma da época de sua origem?

    A evolução da WEB

    WEB 1.0

    • As páginas eram estáticas, ou seja, só permitia a leitura dos documentos dispostos na internet. Não era possível para o usuário fazer cadastro ou modificar o conteúdo da página.

    WEB 2.0

    • É a que nós usamos atualmente. O usuário, pode enviar e receber conteúdo, interage com os elementos e há mais dados sendo envviado hoje em dia do que nunca esreve.

    WEB 3.0

    • A WEB 3.0 tem como objetivo organizar as informações aleatórias que os usuários postam na internet e usar isso para facilitar e agilizar a vida das pessoas.

    Composição da WEB

    • URL x URI
    • HTTP
    • HTML

    URL x URI

    • A URI é um Identificador Uniforme de Recursos e é uma cadeia de caracteres compacta que identifica ou denomina um recurso na Internet.
    • A URL é um Localizador Uniforme de Recursos, então além de identificar, ele precisa apontar onde está o recurso.

    HTTP

    • Significa (Protocolo de Transferência de Hipertexto);
    • É um protocolo de comunicação utilizado para sistemas de informação de hipermídia;
    • Ele funciona enviando uma requisição através de um header e espera uma resposta do servidor.

    HTML

    • Consiste em uma linguagem de marcação utilizada para produção de páginas na web;
    • Podem ser lidos em praticamente qualquer dispositivo e transmitidos pela internet.
